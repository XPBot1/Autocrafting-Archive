An automatic decompactor device that processes compacted blocks input through a top hopper and outputs the decompacted items through a bottom hopper, decompacting only as needed.

Use cases: logs to planks, bone blocks to bonemeal, gold blocks to ingots, etc.
## Features
- Only decompacts what is necessary, locks when output is full
- Works with random input & output
- Compatible with all 1x1 recipes except 1 to 1 like buttons for example
## Considerations
- The bottom hopper and barrel can be replaced with a dropper if needed, however it operates short of 2x hopper speed only outputting 17k items per hour.
