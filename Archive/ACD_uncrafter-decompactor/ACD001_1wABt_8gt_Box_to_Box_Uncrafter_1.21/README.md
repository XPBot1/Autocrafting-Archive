A device that crafts any single item recipe at 8gt speed, mainly useful for uncrafting or decompacting blocks. Input and output are both boxes, allowing for decompacting at 9x HS. Designed as a 'Virtual Bulk' to produce decompressed items on demand from block form storage.
## Features
- Overflow Protected, stops crafting when bottom chest is full
- Crafts single item recipes at 8gt speed
- Box to box: Up to 9x HS decompacting
- Can be disabled via lever
## Considerations
- Multiple toggle states
- Buffers one box
## Notes
This contraption has multiple toggle stats so don't even think about using it it without an Chunk loader, also be prepared to rebuild it if it ever breaks. Also, make sure the Loader Box Dispenser is not updated and becomes QC powered or update any dust  once build.
