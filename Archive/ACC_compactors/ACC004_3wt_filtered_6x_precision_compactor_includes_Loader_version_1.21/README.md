Simple SS3 filtered compactor with optional Loader version if needed.
## Features
- No prefill required
- Box Loader Version available
## Considerations
- no SSI possible
- 3 dusts (but lag is mitigated by green SS2 Line)
- No crafter recovery (needs to be cleaned out manually if crafter overflows)
## Notes
We still suggest you build https://discord.com/channels/856232076252282890/1304491479691034715 if you need more than 3x of speed. also having a loader at each compactor is mostly unnecessary since the hopper speed gets reduced by 9 allowing a 162/h item stream satisfy a single 2x loader
