This crafter uses a two-stage crafter to first convert nuggets into ingots and then into blocks. It can also be used for other items, such as ice.
## Features
- Two-stage crafter: nuggets to ingots, then ingots to blocks
- Can be used for other items such as ice
- Two versions for the block crafting part: SS9 Detection Version, Prefilled Version (requires 9 stacks of gold ingots)
## Compatibility
1.21+
