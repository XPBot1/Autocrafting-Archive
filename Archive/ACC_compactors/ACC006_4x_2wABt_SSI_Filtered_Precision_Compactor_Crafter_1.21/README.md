Can be used for recipes with 8 or 9 items.
## Features
- Low Threshold SSI Filter: It uses SSI filter with minimal Threshold and Prefill.
- Precise Crafting: These crafters only start crafting when they detect a completely full SS9 crafter.
- Global SS11 Line with Recovery Function: The system uses a global SS11 line to control the crafters. It can empty an overflowing crafter that has stopped by pulsing an SS15 signal.
## Instructions
1. Setup filters according to [ACG001 How to setup SSI filters](https://discord.com/channels/856232076252282890/1476712442703773726)
2. Input items using a waterstream
