Compacts a full box of items at 8x hopper speed. It converts them into their compressed form and outputs them into a new box that contains 3 stacks of items.
### How it Works:
The compactor always pushes 8 items before sending a crafting pulse. This ensures that only valid 3x3 recipes are processed, as a recipe with 8 items is always invalid, preventing incorrect crafting attempts.
## Features
- 8x hopper speed compacting
- Compacts up to 37.5 boxes per hour
- Recycles empty boxes
## Considerations
- Input must be only full boxes
- Only works with items that can be compacted in a 3x3 grid
## Instructions
1. Put a full boxes with items you wish to compact in the hopper input (top of the device). It's important that only full boxes are input.
## Compatibility
1.21+
