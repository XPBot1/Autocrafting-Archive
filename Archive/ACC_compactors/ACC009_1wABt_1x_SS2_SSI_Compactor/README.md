Crafts filtered items. Works with 2-9 slot crafts. Can be adapted to be 1 slot. 1 slot and hopper locked variations are in the thread.
## Features
- Compact
- Signal Strength Isolated
## Considerations
- Expensive
- AB tileable
## Instructions
1. The lower filter hopper must be configured to output signal strength 1 when read from the comparator. This comparator subtracts from the comparator reading the top filter hopper to make the filter circuit smaller.
## Notes
Includes hopperlocked and 1 Slot / Uncrafter clocked versions.
## Compatibility
1.21+
