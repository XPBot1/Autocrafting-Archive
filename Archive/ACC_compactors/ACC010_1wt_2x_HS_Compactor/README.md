This compactor prevents double firing of the crafter by preventing a block update to the dropper. This is achieved by powering the noteblock on the second pulse.
## Features
- Can be used for recipes with 4-9 items.
- Prevents double firing of the crafter by preventing a block update to the dropper.
- Achieved by powering the noteblock on the second pulse.
## Notes
The composter is there to make the dropper silent by placing any non-compostable item into the dropper.
## Compatibility
1.21+
