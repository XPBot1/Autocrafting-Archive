A device that crafts shears from iron blocks on demand, producing up to 6200 shears per hour. One double chest of iron blocks can yield 15,552 shears (equivalent to 3.7 million uses).
## Features
- On-demand crafting (only crafts shears when needed)
- Iron block runout protection
- Random input and output proof
- Produces 6200 shears per hour
## Instructions
1. When building, before placing the observer next to the dust, create the shears recipe in the lower crafter with an equal amount of iron ingots (e.g., 32).
2. Ensure the dust is placed as a single dot.
