Takes in boxes of gold blocks and boxes of carrots, and the crafter station outputs boxes of golden carrots.
## Features
- auto stops if it runs out of ingredients
- recycles empty boxes from the carrot unloader.
- spam proof input
- integrated box loader
## Considerations
- Multiple toggle states, make sure all lamps & rs blocks are in the correct state after building
- requires at least a stack of gold blocks and 20 stacks of carrots to start the crafter
- no overflow protection
## Instructions
1. Build the device using the provided Litematic. Look at the image attached below to see what blocks need to be prefilled.
2. Ensure all lamps and redstone blocks are in the correct state after building.
3. Input boxes of gold blocks and carrots to start crafting. When starting the device up for the first time, wait for 20 stacks of carrots to fill up some of the hoppers and droppers (this should take under 9 minutes).
4. Both copper bulbs are off means enough of gold blocks and carrots have been provided, and you can start the device.
5. The device will automatically output boxes of golden carrots.
6. Access empty gold block boxes from the barrel via the trapdoor at the top.
