An automated crafting device designed for recipes with a 4:5 ratio, primarily useful for crafting Sea Lanterns in a guardian farm.
## Features
- Supports crafting any recipe with a 4:5 ratio
- Four crafters to one box loader, 7.2 minutes per box.
- Consumption rates: 18k/h prismarine crystals and 14k/h prismarine shards
- Production rate: 3.6k/h sea lanterns
## Notes
In the non-minimal prefill version, items can easily skip empty hoppers.
## Compatibility
1.21+
